# Описание проекта

В рамках задачи реализован Телеграмм-бот для прогнозирования котировки по ценным бумагам, обращающимся на Американской фондовой бирже.

Для прогнозирования цены использовались 3 модели:

1. K-ближайших соседей (классическое ML);
2. ETS (статистическая модель);
3. LSTM (нейросетевая модель).

Для каждой моджели вычисляется ошибка RMSE (так как хоти максимально обезопасить сделку от неверного решения) и выбирается лучшая модель для прогнозирования.

# Локальный старт приложения
Для локального развёртывания решения. Необходимо загрузить папку app на локальную машину.
Для работы, необходимо указать токен Телеграмм-бота в файле docker-compose.yml на строке 7:
```
1.  services:
2.    server:
3.     build:
4.       context: ./
5.     network_mode: "host"
6.     environment:
7.       - TELEGRAM_TOKEN=
8.     volumes:
9.       # для логов
10.      - ./log:/log
```
После перейти в папку app и в командной строке ввести:
### MacOS

```
docker compose up
```

### linux

```
docker compose up
```
